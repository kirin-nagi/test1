<h2>環境構築</h2>
<br>

<h3>Dockerビルド</h3>
<br>

１，git clone リンク　


２．docker-compose up -d --build　
<br>




<h3>Laravel環境構築</h3>　
<br>

１，docker-compose exec php bash　　


２，composer install　


３，.env.exampleファイルから.envを作成し、環境変数を変更　


４，php artisan key:generate　


５，php artisan migrate　





<h2>使用技術</h2>　
<br>

１，Laravel 8.83.8　　


２，PHP 8.4.4　


３，MySQL 8.0
